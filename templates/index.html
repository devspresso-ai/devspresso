Devspresso

Devspresso is an open-source project that allows you to write and modify code in your browser while supported by a large language model. The intent of the project is to remove manual code editing as much as possible, and instead to let the model do the work. The interaction pattern is meant to approximate a code review, where you are the reviewer and the model the code submitter. Contributions are welcomed at the GitHub repository.

You will need an OpenAI API key to use this service. Alternate integrations with different service providers (and possibly self-hosted open source solutions) are in the works. See a demo of the project in action.
